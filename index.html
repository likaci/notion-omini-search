Direct search in Notion from Chrome address bar

Search Notion Faster Than Ever

Transform your Chrome address bar into a powerful Notion search tool. No more switching tabs or navigating through Notion's interface - search directly from your browser's address bar!

Setup Instructions

Open Chrome Settings
Go to Chrome Settings ‚Üí Search engines ‚Üí Manage search engines and site search
Add Site Search
Click Add to create a new Site Search
Configure Search Engine
Fill in the details:
Search engine: Notion Search
Shortcut: ns
URL: https://www.notion.so/?nos=%s
Pro Tip: For searching in a specific workspace, use:
https://www.notion.so/WORKSPACE_ID?nos=%s

How to Use

Type the shortcut
In Chrome address bar, type: ns
Activate search
Press Tab or Space
Enter your search
Type your search keyword
Search!
Press Enter and watch the magic happen
